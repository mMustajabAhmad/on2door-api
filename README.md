# On2Door API

A real-time driver management and delivery tracking backend built with Rails.
---
## Setup Instructions
### Requirements
- **Ruby**: 3.3.3
- **Rails**: 7.2 (API mode)
- **PostgreSQL**: 16
---
### :gear: Installation
1. Clone Project
```sh
git clone https://github.com/DevDenCo/on2door-api.git
```
2. Project Directory
```sh
cd on2door-api
```

3. Install ruby version 3.3.3
```sh
rbenv install 3.3.3
```
```sh
ruby -v
```
4. Install Gems
```sh
bundle install
```
5. Steup Database
```sh
rails db:create
rails db:migrate
```
6. Start Rails Server
```sh
rails server
```
